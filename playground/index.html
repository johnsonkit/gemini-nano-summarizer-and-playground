How to Enable Experimental features in Google Canary
  1. Download the latest version of Google Canary
  2. Type chrome://flags in the address bar
  3. Enabled without language pack limit "Experimental translation API"
  4. Enable "Prompt API for Gemini Nano"
  5. Enable "Summarization API for Gemini Nano"
  6. Enable "Writer API for Gemini Nano"
  7. Enable "Rewriter API for Gemini Nano"
  8. Enabled BypassPerfRequirement "Enables optimization guide on device" (It will download the Gemini Nano model in the background)
  9. Restart the browser
  10. Type chrome://components in the address bar and update "Optimization Guide On Device Model"
